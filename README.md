# 🎨 Rajpoot Custom WordPress Theme

A professional custom WordPress theme built from scratch with PHP, HTML, CSS & JavaScript.

## 🚀 Features

- Responsive Design (Mobile Friendly)
- Custom Hero Section with Animated Coder Cartoon
- Skills Section with Progress Bars
- Projects Showcase Section
- Blog Posts Section
- Custom Header & Footer
- Mobile Menu with Overlay
- Floating Tech Badges Animation
- Single Post Page with Sidebar
- Widget Areas (Recent Posts, Categories)

## 🛠️ Tech Stack

- **PHP** - WordPress theme development
- **HTML5** - Semantic markup
- **CSS3** - Custom styling & animations
- **JavaScript** - Mobile menu & interactions
- **WordPress** - CMS platform

## ⚙️ Installation

1. Download/clone this repository
2. Copy `rajpoot-theme` folder to `wp-content/themes/`
3. Go to WordPress Dashboard → Appearance → Themes
4. Activate **Rajpoot Theme**
